Documentación del código – Parpadeo de LED
¿Qué hace el programa?

Este código enciende y apaga un LED conectado al pin 13 de Arduino de forma repetitiva.

Componentes
Arduino
LED (conectado al pin 13)
Funcionamiento
1. Configuración
Se define el pin 13 como salida.
2. Ejecución
El LED se enciende (HIGH).
Espera 1 segundo.
El LED se apaga (LOW).
Espera 1 segundo.

Este proceso se repite continuamente.
